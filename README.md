# My Portfolio Website

A responsive portfolio website built with **HTML**, **CSS**, and **JavaScript** to showcase my skills, projects, and professional background.

## 📋 Overview

This portfolio website is my first web development project, demonstrating my passion for creating modern and user-friendly digital experiences. It serves as a platform to introduce myself and display my work in web development and programming.

## 🎯 Features

- **Responsive Design** - Clean and professional layout that works on all devices
- **About Section** - Introduction and background information
- **Projects Showcase** - Display of completed projects with descriptions
- **Skills Section** - List of technical skills and competencies
- **Contact Form** - Interactive form for visitors to reach out
- **Navigation** - Easy-to-use navigation buttons for different sections

## 💻 Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Styling and responsive design
